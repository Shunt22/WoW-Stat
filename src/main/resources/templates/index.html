Hello!

This is my first "useful" website. It was created just to improve my Java, HTML and JavaScript skills.



This site collects and shows some statistics from the WoW(World of Warcraft) game.

Data comes from EU PvP leaderboard (2v2, 3v3 and RBG) which is provided by WoW API by Blizzard Entertainment.

Things are being used:
  • Server-side
    • Spring framework
    • Thymeleaf
  • Client-side
    • Flot charts
    • jQuery


Thanks for visiting.
